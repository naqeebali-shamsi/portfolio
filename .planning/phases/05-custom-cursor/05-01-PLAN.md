<task type="auto">
  <name>Task 1: Create CustomCursor component and CSS cursor:none rule</name>
  <files>src/components/CustomCursor.tsx, src/styles/base.css</files>
  <action>
Create `src/components/CustomCursor.tsx`:

1. **Element:** A single `div` with `ref`, positioned `fixed top-0 left-0`, `pointer-events-none`, `z-[9999]`, `-translate-x-1/2 -translate-y-1/2`, `rounded-full`, `border-[1.5px] border-current`, `mix-blend-difference`. Initial inline styles: `width: 24px, height: 24px, visibility: hidden, color: #1a1a1a`.

2. **useGSAP with matchMedia:** Wrap entire cursor system in `gsap.matchMedia()` with two conditions:
   - `desktop`: `(pointer: fine) and (min-width: 768px) and (prefers-reduced-motion: no-preference)`
   - `desktopReduced`: `(pointer: fine) and (min-width: 768px) and (prefers-reduced-motion: reduce)`

   Use `context.conditions!` to determine which matched. Both conditions activate the cursor; `desktop` enables animated morphs, `desktopReduced` uses `duration: 0` for morphs.

3. **On match enter:**
   - `gsap.set(el, { autoAlpha: 0.5 })` to show the ring at rest opacity
   - Add `custom-cursor-active` class to `document.documentElement`
   - Create `xSet = gsap.quickSetter(el, "x", "px")` and `ySet = gsap.quickSetter(el, "y", "px")`

4. **mousemove handler:** Call `xSet(e.clientX)` and `ySet(e.clientY)` -- instant follow, no lerp.

5. **Morph state machine** via `morphTo(type)` function:
   - `"link"`: width/height 48px, borderWidth 1.5, backgroundColor `rgba(26,26,26,0.08)`, opacity 1
   - `"project"`: width/height 80px, borderWidth 1, opacity 1
   - `"text"`: width/height 18px, opacity 0.6
   - `null` / default: width/height 24px, borderWidth 1.5, backgroundColor `transparent`, opacity 0.5

   All morphs use `duration: desktop ? 0.2 : 0` and `ease: desktop ? "back.out(2)" : "none"`.

6. **mouseover handler (event delegation):** On `document`, use `(e.target as HTMLElement).closest("[data-cursor]")` to find morph target. Call `morphTo(target?.getAttribute("data-cursor") ?? null)`.

7. **Click feedback:**
   - mousedown: `gsap.to(el, { scale: 0.8, duration: 0.1, ease: "power2.in" })`
   - mouseup: `gsap.to(el, { scale: 1, duration: 0.25, ease: "back.out(3)" })`

8. **Window exit/enter:**
   - `document.documentElement` mouseleave: `gsap.to(el, { autoAlpha: 0, duration: 0.15 })`
   - `document.documentElement` mouseenter: `gsap.to(el, { autoAlpha: 0.5, duration: 0.15 })`

9. **Cleanup (return function):** Remove `custom-cursor-active` class, remove all 6 event listeners (mousemove, mouseover, mousedown, mouseup, mouseleave, mouseenter).

**Add CSS rule to `src/styles/base.css`** (inside the existing `@layer base` block, after the `button` rule):

```css
.custom-cursor-active,
.custom-cursor-active *,
.custom-cursor-active *::before,
.custom-cursor-active *::after {
  cursor: none !important;
}
```

**Do NOT:**
- Use `useState` for cursor position (causes 60+ re-renders/sec)
- Use CSS transitions for position (layout thrashing)
- Use `pageX/pageY` (offset when scrolled -- use `clientX/clientY` since element is `position: fixed`)
- Mount/unmount different elements for morph states
- Add `pointer-events` to anything other than `none` on the cursor element
